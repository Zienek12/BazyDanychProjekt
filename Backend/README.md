# Online Food — Spring Boot + MySQL + Docker Compose

Projekt **Online Food** to aplikacja backendowa stworzona w **Java 21** i **Spring Boot 3.5**, umożliwiająca zarządzanie użytkownikami, restauracjami i zamówieniami w systemie zamawiania jedzenia online.  
Projekt jest w pełni zdockerowany i korzysta z **MySQL**.

---

## Technologie

- Java 21  
- Spring Boot 3.5  
- Spring Web    
- MySQL 8  
- Docker & Docker Compose  
- Maven  

---

## Uruchamianie aplikacji przez Docker Compose

### Wymagania
- Docker Desktop (Windows, Mac) lub Docker Engine (Linux)  
- Docker Compose v2

### Uruchomienie

W katalogu głównym projektu:

docker compose up --build

## Po uruchomieniu:

 - Backend: http://localhost:8080

 - MySQL host: localhost:3307

 - Database: online_food

 - Username: root

 - Password: root
